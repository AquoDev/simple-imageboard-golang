Simple Imageboard

Quick Guide

GET

/index /page/[0-9] /thread/[1-9][0-9]* /post/[1-9][0-9]* /health /settings

POST

/post Header:
- Content-Type: application/json; charset=UTF-8

Body:
- content: [required, type: string, min: 1, max: 1000] <post content>
- reply_to: [optional, type: uint64] <post id you reply to> OR null to make a new thread
- pic: [optional, type: string, max: 512] <direct external image url> OR null
- delete_code: [optional, type: string, max: 128] <password to delete the post any time> OR null to make a random one of 32 characters long

DELETE

/post Header:
- Content-Type: application/json; charset=UTF-8

Body:
- id: [required, type: uint64] <post id>
- delete_code: [required, type: string] <code received after successful posting>

WARNING:
- if your post started a parent thread, every reply will also be deleted
