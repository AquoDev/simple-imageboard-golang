Simple Imageboard

Quick guide

GET

/index /thread/[1-9][0-9]* /post/[1-9][0-9]* /health /settings /robots.txt

POST

/post Header:
- Content-Type: application/json; charset=UTF-8

Body:
- content: [required, type: string, min: 1, max: 1000] <post content>
- reply_to: [optional, type: uint64] <post id you reply to> OR null to make a new thread
- pic: [optional, type: string, max: 512] <direct external image url> OR null
- password: [optional, type: string, max: 128] <password to delete the post at any time> OR null to make a random one of 32 characters long
- sage: [optional, type: boolean] <don't bump parent thread when it's set to true> OR false OR null

DELETE

/post Header:
- Content-Type: application/json; charset=UTF-8

Body:
- id: [required, type: uint64] <post id>
- password: [required, type: string] <password received after making a post> OR <admin password>

WARNING:
- if your post started a parent thread, every reply will also be deleted

PUT

/post Header:
- Content-Type: application/json; charset=UTF-8

Body:
- id: [required, type: uint64] <post id>
- password: [required, type: string] <admin password>
- sticky: [required, type: boolean] <put this thread at the top when it's set to true> OR <make this thread into a regular one when it's set to false OR null>
- closed: [required, type: boolean] <forbid replies when it's set to true> OR <allow replies when it's set to false OR null>

WARNING:
- this is not supposed to be used by regular users
